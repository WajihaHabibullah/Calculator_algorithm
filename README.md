# Calculator_algorithm
Step 1: Start.\
Step 2: Input num1, num2, opt.\
Step 3: Calculate a.\
Step 4: If opt = + then a = num1 + num2\
Step 5: If opt = - then a = num1 - num2\
Step 6: If opt = * then a = num1 * num2\
Step 7: If opt = / then a = num1 / num2\
Step 8: If opt = % then a= num1 % num2\
Step 9: Print a.\
Step 10: Stop.
